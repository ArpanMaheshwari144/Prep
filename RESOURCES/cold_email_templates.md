# 📧 Cold Email Templates (Outreach Patterns)

> **Use case:** LinkedIn referrals, recruiter outreach, hiring manager direct, founder/CEO emails
> **Source pattern:** Niraj Pant → Evan Spiegel (2014, viral example)
> **Reality check:** 100 emails → 5 replies → 1-2 outcomes (not 1-email magic)

---

## ⚠️ Mindset First (BEFORE templates)

```
ALWAYS REMEMBER:
   • Single success stories hide BTS context
   • Network, privilege, timing, luck = invisible
   • Pattern improves probability — doesn't guarantee
   • Volume + Quality + Right targets = formula
   
DON'T:
   ❌ Expect 1 email = job
   ❌ Get demoralized when 95% no reply
   ❌ Copy-paste exact same email everywhere
   
DO:
   ✅ Use pattern, customize per recipient
   ✅ Track sent/replies/outcomes
   ✅ 50-100 emails minimum before evaluating
```

## 🎯 The "Niraj Pant" Pattern (Hall of Fame)

### Original (2014, to Evan Spiegel):
```
Subject: Internship

Hey,

My name's Niraj Pant.

I understand your time is valuable. I'll only write three bullet points.

Programming since 8th grade.

Have most experience working in Java/Obj-C/Android/iOS.

Want to intern for Snapchat this summer as a high school junior. How?

Thanks,
Niraj
```

### Why It Worked:
```
1. BREVITY ANNOUNCEMENT → respect signal
2. CREDIBILITY (1 line)  → "since 8th grade"
3. RELEVANT SKILL        → exact tech stack match
4. SHARP DIRECT ASK      → specific + question = forces engagement
5. ZERO FLUFF            → no "hope you're well" / "love your work"
```

## 📋 STEAL-ABLE TEMPLATE (3-Bullet Rule)

```
Subject: [2-3 word specific topic]

Hi [Name],

[1-line brevity respect — optional]

• [Credibility hook — 1 line]
• [Relevant experience/skill — 1 line]
• [Direct specific ask + how/can/would]

Thanks,
[Your Name]
```

## 🎯 Variations (use case wise)

### A) **LinkedIn Referral Ask** (warm-ish contact)

```
Hi [Name],

Saw you're at [Company] — I'll keep this short.

• 4+ years Java backend (Spring Boot, microservices, JWT)
• Production debugging + on-spot client crisis resolution at Konovo
• Moving to Bangalore June 15 — would your team consider a referral 
  for backend roles?

Happy to share resume if useful.

Thanks,
Arpan
```

---

### B) **Cold Email to Hiring Manager** (no warm intro)

```
Subject: Backend SDE — referral interest

Hi [Name],

Quick 3-line pitch:

• 4+ years Java backend dev (Spring Boot, MySQL, JPA, JWT/OAuth)
• Strong production debugging — on-spot client issue resolution 
  experience at Konovo Inc
• Recently built end-to-end Spring + Docker + MySQL project; 
  moving to Bangalore June 15

Could you suggest the right channel for backend roles at [Company]?

Thanks,
Arpan
LinkedIn: linkedin.com/in/[handle]
```

---

### C) **Recruiter Reachout**

```
Hi [Recruiter],

Saw [Role] open at [Company] — quick fit check:

• 4+ years Java/Spring, microservices in production
• Experience: [1-2 specific things relevant to JD]
• Available to interview within [timeline]

Would love to chat — 15-min call this week?

Thanks,
Arpan
```

---

### D) **CEO/Founder Direct** (highest bar — go big)

```
Subject: [Specific company problem you solve]

Hi [Founder Name],

Your team's working on [specific project from public info].

3 reasons I might be useful:

• [Specific skill that addresses their problem]
• [Track record evidence — concrete metric]
• [Why you specifically — unique angle]

Would 10 min next week work? Happy to come in or do video.

Thanks,
[Name]
```

---

### E) **Re-engagement (no reply 1 week later)**

```
Hi [Name],

Bumping this — understand calendars are full.

[1 line — refreshed pitch or new angle]

If now isn't right, happy to check back in [X weeks].

Thanks,
Arpan
```

## 🔥 Subject Line Patterns

```
✅ GOOD subjects:
   "Backend SDE — quick fit check"
   "3-bullet pitch (Java/Spring)"
   "Referral interest — [Company] backend"
   "Niraj Pant intro" (mutual contact name)
   
❌ BAD subjects:
   "Hi" / "Hello" / "Job"
   "I would love to discuss opportunities"
   "Looking for opportunities"
   "Resume attached"
   "URGENT" / ALL CAPS
```

## 🎤 Behavioral Q During Call

When email gets reply + call:

**"How did you find me?"**
> *"Saw on LinkedIn — your team's tech stack matches my background, wanted to explore fit before applying through portal."*

**"Why this company?"**
> *"[1 specific reason from public info — recent product, tech blog post, specific role's challenge]"*

**"Why now?"**
> *"Moving to Bangalore June 15, exploring backend SDE roles. Looking for [tech depth / scale / mentorship — pick honest one]."*

## 💎 Mindset Anchors

```
1. Volume game — not perfection game
2. Personalization > template (1-line custom per recipient)
3. Direct ask > "would love to chat"
4. Brevity is signal of seniority
5. Reply rate ≠ rejection — silence is data, not personal
6. Track + iterate (which version gets replies?)
7. BTS hidden for viral stories — yours is YOUR data, trust YOUR effort
```

---

## 🚫 Cold Email Anti-Patterns

```
❌ "Hope this email finds you well"          → time waste
❌ "I'd love to discuss opportunities"        → vague
❌ "I'm a passionate engineer"                 → buzzword
❌ "Please let me know if interested"          → passive
❌ Multi-paragraph essay                       → read = NEVER
❌ Generic copy-paste (no name customization)  → spam folder
❌ Asking for everything                        → "any role"
❌ Resume attached without text body            → low effort
❌ All caps subject                             → spam filter
❌ Follow-up daily for week                    → harassment
```

## ⚡ Quick Reference

```
TEMPLATE STRUCTURE:
   1. Custom opener (1 line — show you researched)
   2. 3 bullets (credibility + skill + ask)
   3. Direct close (specific, time-bound if possible)

LENGTH:
   • Email: ~80 words MAX
   • LinkedIn: ~50 words (smaller window)

TIMING:
   • Tuesday-Thursday
   • Mid-morning (10-11 AM their timezone)
   • Avoid Mondays/Fridays/holidays

FOLLOW-UP:
   • +7 days (1 bump)
   • +14 days (move on, archive)
   • Don't desperate-spam
```
