# LinkedIn Referral Message Templates
### By Arpan Maheshwari | Copy paste kar, naam badal, bhej

---

## RULES (yaad rakh):
```
1. SHORT rakh — 3-5 lines max. Lamba = ignore.
2. PERSONALIZE — company naam, role naam mention kar.
3. RESUME attach mat kar pehle message mein — pehle baat, phir maange toh de.
4. "Thanks either way" — pressure mat daal.
5. Business hours mein bhej (9AM-5PM unke timezone).
6. 1 din baad follow up — 1 week baad nahi.
```

---

## TEMPLATE 1: Connection Request Note (300 chars limit)

**Kab use:** Pehli baar connect kar raha — note ke saath.

```
Hi [Name], I'm a Java Backend Engineer (4+ yrs, Spring Boot). 
Applied for [Role] at [Company]. Would you be open to referring 
me? Happy to share resume. Thanks!
```

**Example:**
```
Hi Chandan, I'm a Java Backend Engineer (4+ yrs, Spring Boot). 
Applied for Backend Engineer role at N26. Would you be open to 
referring me? Happy to share resume. Thanks!
```

## TEMPLATE 2: After Connection Accepted — First Message

**Kab use:** Connect accept ho gaya, ab detailed message bhej.

```
Hi [Name], thanks for connecting!

I'm Arpan, a Java Backend Engineer with 4+ years building 
microservices for a healthcare platform across 10+ countries. 
I've resolved 500+ production incidents and been recognized 
by VP and CTO three times.

I applied for the [Role] role at [Company] and I'm really 
excited about the opportunity. Would you be open to referring 
me? I'd be happy to share my resume.

Totally understand if not — appreciate your time either way!

Best,
Arpan
```

---

## TEMPLATE 3: After Rejection — Asking for Reconsideration

**Kab use:** Company se reject ho gaya, lekin andar ka banda connect hai.

```
Hi [Name], thanks for connecting!

I'm Arpan — Java Backend Engineer, 4+ yrs, Spring Boot. 
I applied at [Company] but got screened out in initial round.

Would really appreciate any tips — is there another role 
you'd suggest, or any way to get reconsidered?

Happy to share my resume if helpful. Thanks!
```

## TEMPLATE 4: Personalized — Unke Profile pe Kuch Dekha

**Kab use:** Jab unke profile mein kuch interesting dikhe — project, post, talk.

```
Hi [Name], I came across your profile and noticed [something 
specific — e.g., "your work on the payments team" or "your 
talk about microservices"]. Really impressive!

I'm a Java Backend Engineer (4+ yrs) and I recently applied 
for [Role] at [Company]. Would love to learn more about the 
team and the opportunity. Would you be open to a quick chat 
or referring me?

Thanks either way!
Arpan
```

**Example:**
```
Hi Gaurav, I came across your profile and noticed your 9+ 
years of Java experience at N26. Really impressive!

I'm a Java Backend Engineer (4+ yrs) and I applied for 
Backend Engineer role at N26. Would love to learn more 
about the team. Would you be open to referring me?

Thanks either way!
Arpan
```

## TEMPLATE 6: Follow-up (1 din baad reply nahi aaya)

**Kab use:** Message bheja, 2-3 din ho gaye, reply nahi.

```
Hi [Name], just following up on my earlier message. 
Completely understand if you're busy — no pressure at all. 
Would really appreciate any help if possible. Thanks!
```

**RULE: Sirf 1 follow-up. Reply nahi aaya toh MOVE ON. Spam mat kar.**

## DO's and DON'Ts

```
DO:
  ✅ Company + Role mention karo (specific)
  ✅ Short rakh (3-5 lines)
  ✅ "Thanks either way" likho (no pressure)
  ✅ Resume BAAD mein bhejo (maange toh)
  ✅ Business hours mein bhejo

DON'T:
  ❌ "Please sir help" — desperate mat lago
  ❌ Lamba essay mat likho — padhenga nahi
  ❌ Resume PEHLE mat attach karo — pushy lagta
  ❌ 3-4 baar follow up mat karo — spam hai
  ❌ "I need a job urgently" — KABHI mat likho
  ❌ Same person ko 2 companies ke liye mat pooch
```
